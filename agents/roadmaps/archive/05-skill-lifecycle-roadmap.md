# Skill Lifecycle Roadmap

**Status: ✅ COMPLETE**

## Goal

Manage versions and evolution of skills.

## Concepts

- versioning (semver)
- deprecation
- usage tracking

## Constraints

- backward compatibility where possible
- no silent breaking changes
- lifecycle metadata must be used by tooling, not just documented

## End state

Skills can be versioned, deprecated, migrated, and cleaned up safely.
