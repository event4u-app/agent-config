# Tool Integration Roadmap

**Status: ✅ COMPLETE**

## Goal

Add controlled MCP / API integrations.

## Key principle

Tools are permissions, not abilities.

## Architecture

- Tool Registry
- Tool Adapters
- Permission Layer

## Constraints

- allowlist only
- deny by default
- no arbitrary execution
- no hidden credentials model

## End state

Skills can declare safe tool usage in a controlled, reviewable way.
