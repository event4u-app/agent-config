# Feedback Loop Roadmap

**Status: ✅ COMPLETE**

## Goal

Continuously improve skills based on usage.

## Concepts

- skill scoring
- feedback signals
- auto suggestions

## Constraints

- no blind automatic changes
- human review remains mandatory
- repeated patterns should become structured improvements

## End state

Repeated problems can be detected, categorized, and promoted into rules, skill changes, or linter improvements.
