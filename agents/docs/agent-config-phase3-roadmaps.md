## Purpose

The package should help evolve the current system from:

> governed, config-first skill framework

toward:

> governed, observable, safely evolving agent capability system

## Important principle

Never add power before control.

Execution, tools, and automation must always respect:

- rules
- skill quality standards
- source-of-truth model
- governance layer
